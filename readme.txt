== Description ==

Universal Theme  adapts your website to mobile devices and the desktop or any other viewing environment. Theme features 9 Page Templates, 11 Widget Areas, 6 Template Layouts, 3 Menu Positions and more. 
WooCommerce Compatible, Multilingual Ready (WPML), RTL-Language Support, Retina-Ready, W3C Markup Validated and currently translated into 34 languages. 
Cross-Browser compatible and yes even the IE7. Info & Support at http://www.universalthe.me/

== Installation ==



**Automatic installation**
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser.
To do an automatic install of **Universal** log in to your WordPress admin panel, navigate to the Themes menu and click Add New.
In the search field type “**Universal**” and click Search.You can install it by simply clicking Install Now.
After clicking that link you will be asked if you’re sure you want to install the theme. 
Click yes and WordPress will automatically complete the installation. Once the theme has been sucessfully installed click Activate.

**Manual installation**
This manual installation method involves downloading our theme and uploading it to your webserver via WordPress uploader.

1.  Download the theme file to your computer.
2.  Log in to your WordPress admin panel, navigate to the Themes menu and click Add New.
3.  Click Upload
4.  Chose the zip file on your computer
5.  Click Install Now
6.  Once the theme has been sucessfully installed click Activate.
 

**FTP Option**
This manual installation method involves downloading our theme and uploading it to your webserver via your favourite FTP application.
Download the theme file to your computer and unzip it
Using an FTP program, or your hosting control panel, upload the unzipped theme folder to your WordPress installation’s wp-content/theme/ directory.
Activate the theme from the Themes menu within the WordPress admin.
**Upgrading**
Automatic updates should work a charm; as always though, ensure you backup your site just in case.
